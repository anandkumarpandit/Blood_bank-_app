## Tech Stack

**Client:** React, contextApi, TailwindCSS

**Server:** Node, Express, MongoDB


### Run Locally

Clone the project

```bash
  git clone  https://github.com/anandkumarpandit/Blood_bank-_app
```

Go to the project directory


Install dependencies for frontend and backend

```bash
  npm install
```

Start the server

```bash
  npm run start
```
