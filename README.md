# Blood bank app

Welcome to LifeLink - Blood Donation Management System, an end-to-end platform designed to optimize donor registration, blood requests, and inventory tracking. The system features an intuitive interface, modern design, and seamless coordination between hospitals, donors, and recipients. With secure authentication, role-based control, and real-time updates, the platform ensures accessibility and operational efficiency. Its responsive architecture enables fast data retrieval and a smooth user experience across all devices, making blood donation management more effective than ever.

Tech Stack
Client: React, ContextAPI

Server: Node, Express, MongoDB

Run Locally
Clone the project

  git clone 
Go to the project directory


Install dependencies for backend and frontend

  npm install
Start the server

  npm start

Start the application

  npm run dev
